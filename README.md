# Installation
You could install requirements for this project with a virtual environment as venv

git clone https://github.com/enomis-dev/SnakeGame

cd SnakeGame

pip install virtualenv

python -m venv venv

venv\Scripts\activate

pip install pygame

# Run

python snake_game.py
